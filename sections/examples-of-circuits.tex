\section* {Examples Of Circuits}


A digital logic circuit is a set of logic gates that takes in signals (usually 0s and 1s) and uses a set rule or function to make an output. In short, circuits are made to do useful things. They take inputs like x, y, and z and give an output. You can use a Boolean expression to describe how a circuit works.

This section has some circuits and shows you how to solve them.

\subsection{EXAMPLE 1}
